# PDS Project - Software Development Project

# About the Project
This is a university group project developed as part of our coursework. 
The "Civic Participation Platform for Incidents" aims to improve communication between citizens and municipal services by providing a centralized and transparent system for reporting urban issues and waste management problems. This platform solves these issues by allowing users to submit geolocated incident reports with images, while enabling administrators to manage and track them efficiently through an interactive system.

# Team Project 
This project was developed in collaboration with a team of students using GitLab for version control, task management, and collaborative development.
